# Composability-Class Matrix

Status: exploratory research under `exploration/`. Not production data, not doctrine, not a change to the live dictionary. Companion to `HANDOFF_STATE_PAPER.md`. It classifies **types** of two-dexterity pairing by their structural composability status, from the grammar and the proposed handoff state alone. It does not enumerate the 64 pairings as valid or invalid, and it makes no physical-realizability claim.

## The two layers of claim, kept separate

Every classification below is tagged by which layer it belongs to:

- **[G-unsupported]** the current grammar does not certify this pairing either way (the encoded state is silent). This is not a claim of impossibility.
- **[M-incompatible]** under the explicit four-component handoff state model (carriage side, reference-foot identity, support-and-free-leg parity, stance mode), this pairing type cannot close. This is conditional on the model, which the grammar does not supply.

Nothing here is tagged impossible; physical realizability is out of scope.

## The classes

| Pairing type | Structural description | Class | Layer |
|---|---|---|---|
| Frame-preserving, leg-consistent | the first dexterity leaves the reference foot unchanged and frees exactly the leg the second dexterity must circle with | definitely composable | certified from encoded structure |
| Frame-transporting | the first dexterity changes which foot sets or supports the bag, so the second dexterity's side is measured in a turned frame | unresolved: composable but re-coordinatized; the address is a bundle point, not a free-product point | [G-unsupported] |
| Leg-conflicting | the second dexterity's circling leg is the one the first dexterity left as support, with no intervening event to free it | cannot close | [M-incompatible] |
| Carriage-reachable | the first dexterity's output carriage is a side the second dexterity can engage | mostly composable: a dexterity's output side is a free choice, so the carriage rarely blocks on its own | certified from encoded structure |
| Stance-gated | composability turns on whether the handoff is planted, suspended, or airborne, which the notation encodes only sometimes | unresolved where the stance is untokenized | [G-unsupported] |
| Hidden-state-dependent | composability turns on vertical phase or trajectory, which the notation does not encode and whose structural-versus-physical status is itself unsettled | unresolved; possibly not a structural question at all | [G-unsupported] |
| Potentially identified | two addresses differ only in an intermediate component the terminal never records (for example the invisible carriage), so they may be one movement | unresolved: candidate identification, not a distinct-pair claim | [G-unsupported] |
| Diagonal (repeated dexterity) | the two dexterities are identical | ambiguous: an ordered pair, or a single dexterity under a count operator | [G-unsupported] |

## What the distribution says

The certifiable classes are narrow. Only the frame-preserving-leg-consistent and carriage-reachable types are composable from the encoded structure, and even those assume the frame did not turn. Exactly one type is incompatible, and only under the explicit model, never from the grammar itself. Every remaining type is unresolved for want of state: the grammar cannot certify it either way. So the matrix's headline is not a ratio of valid to invalid pairings; it is that **the encoded state decides very little**, and the two-dex space is dominated by pairings whose composability is a question the current grammar cannot pose, let alone answer.

This is the constructive form of the falsification. The strict product failed not because many pairings are forbidden but because the grammar cannot tell composable from incompatible for most of them. The count 64 is an upper bound precisely because the unresolved classes might contain both identifications (fewer distinct movements) and model-incompatibilities (fewer valid movements), and the grammar fixes neither.

## Discipline honored

No pairing is called impossible because the notation cannot certify it. The single [M-incompatible] type is explicitly conditional on a state model the grammar does not carry. The diagonal is left ambiguous rather than forced. Scoring cross-terms (the far-receiver grade) are documented in the handoff paper and are not used here as evidence of structural incompatibility.
