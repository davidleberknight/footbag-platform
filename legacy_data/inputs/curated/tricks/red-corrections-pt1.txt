The very first thing since you asked this question a lot; are any of these tricks still used in competition and outside it? The answer is yes, they all are. There are very few tricks that freestylers don't do anymore. Everything on this list is still in play to this day :)
*Q1.1 — Atom Smasher: is it 3 ADD or 4 ADD?**
Atom Smasher is a 4 ADD move. It involves an element called X-Dex which along with the dexterity ADDs make it a 4 ADD move.

Flurry is a 4 ADD move. It has three dexes and a one ADD stall, to equal 4 ADDs.

Sidewalk is same side stepping butterfly, which is 4 ADDs. It should be added.

Tombstone is a stepping drifter. It should be added too.

Flail and Omelette are common names and should be added.

Merkon is a standalone name/trick.

Terrage and Barrage are standalone names. But they can also be used to add onto a trick, such as barraging paradox mirage (which is called Fury).

Royale and Ripstein - primary entries. These are popular tricks.

Eclipse: is the jump a scored body element?** - Yes, in a manner of speaking. The hop over portion counts for an ADD. It is a 3 ADD move.
