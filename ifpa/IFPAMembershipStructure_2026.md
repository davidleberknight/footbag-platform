# IFPA Membership Rules

Status: current IFPA membership rules for the modernized footbag.org era.

These rules replace the old annual-membership model and control current IFPA membership policy unless IFPA adopts a later rule.

## 1. Membership Tiers

All IFPA memberships are lifetime memberships.

The current membership tiers are:

- **Tier 0: Registered Member**. Free registered account.
- **Tier 1: IFPA Member**. Purchased membership. Default price USD $10.
- **Tier 2: IFPA Organizer Member**. Purchased organizer membership. Default price USD $50.
- **Tier 3: IFPA Director**. Director or board-level status assigned by IFPA governance.

There are no annual IFPA memberships.

Tier 0 members may purchase Tier 1 or Tier 2 directly.

Hall of Fame induction grants Tier 2.

BAP induction grants Tier 2.

Tier 3 is a governance status. When Tier 3 status ends, the member returns to their underlying membership tier.

If a Tier 0 member becomes Tier 3, their underlying membership tier becomes Tier 1.

## 2. Active Player Status

Active Player is a temporary status for Tier 0 members only.

Active Player status gives a Tier 0 member Tier 1 benefits while current.

Active Player status lasts 730 days from the most recent qualifying event attendance, vouch, or one-time club-join grant.

A later qualifying event attendance or vouch extends Active Player status.

Active Player status does not change membership tier.

If a Tier 0 Active Player becomes Tier 1 or Tier 2, Active Player status ends.

If a Tier 0 Active Player becomes Tier 3, their underlying membership tier becomes Tier 1 and Active Player status ends.

## 3. Qualifying Events and Club Participation

### 3.1 Officially Registered Events

Attendance at any event officially registered through the IFPA website grants or extends Active Player status for Tier 0 members.

The 730-day period normally runs from the event end date.

For a single-day event, the event date is used.

If the event end date is unknown, the event start date is used.

An older event must not shorten an existing later Active Player expiry date.

### 3.2 One-Time Club-Join Grant

A Tier 0 member who has never previously been an Active Player may receive one 730-day Active Player period when they first join an IFPA club.

This is a one-time grant.

Joining additional clubs does not grant additional Active Player periods.

The club-join grant does not change membership tier.

## 4. Vouching

Tier 2+ members may vouch for Tier 0 members to receive or extend Active Player status.

A vouch grants or extends Active Player status only.

A vouch does not change membership tier.

An older vouch must not shorten an existing later Active Player expiry date.

## 5. Hall of Fame and BAP

Hall of Fame and BAP honors remain separate honors.

Hall of Fame induction grants Tier 2.

BAP induction grants Tier 2.

## 6. Official IFPA Roster

The Official IFPA Roster includes:

- Tier 1 members
- Tier 2 members
- Tier 3 members
- Tier 0 members with current Active Player status

Tier 0 members without current Active Player status are not included.

Roster records should distinguish membership tier from Active Player status.

The Official IFPA Roster is not public.

Tier 2+ members may access the Official IFPA Roster for official IFPA event and organizer purposes.

## 7. Clarifications

If a Tier 0 Active Player buys Tier 1 or Tier 2, Active Player status ends.

If Active Player status expires for a Tier 0 member, Tier 1 benefits and roster inclusion end.

If a Tier 1+ member attends a qualifying event, joins a club, or is vouched for, Active Player status is not needed.

If Tier 3 status ends, the member returns to their underlying membership tier: Tier 1 if they entered Tier 3 from Tier 0, Tier 1 if they entered from Tier 1, or Tier 2 if they entered from Tier 2, Hall of Fame, or BAP.
