# Freestyle exploration — index

This tree is investigation scratch space: audits, evidence packets, corpora, and
one-off analyses produced while building the freestyle encyclopedia. It is not the
system of record. Doctrine lives in `freestyle/doctrine/`, the plan of record is
`IMPLEMENTATION_PLAN.md`, and the content authorities are named in
`freestyle/doctrine/AUTHORITY.md`.

Completed work is retired to `_archive/` (by month; the latest retirement pass is
in `_archive/2026-07/`). Everything still in the live tree is here because a
doctrine document, a skill, the pipeline code, the plan, or a test still points at
it, or because the work is genuinely unfinished. The groups below say which.
