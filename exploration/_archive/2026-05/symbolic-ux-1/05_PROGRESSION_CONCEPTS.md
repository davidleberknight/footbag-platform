# SUX-1 Task E — Educational Progression Concepts

Three progression sequences showing how a learner can climb from foundation primitives to complex compounds via symbolic-grammar-driven pathways.

**Date:** 2026-05-12

---

## Progression rendering grammar

Each progression uses consistent visual grammar:

```
PROGRESSION TITLE
│
├── Foundation:  anchor primitive (with ADD)
│
├── Tier 1:      single-modifier compounds (with ADD)
│
├── Tier 2:      multi-modifier or variant compounds
│
└── Flagship:    highest-difficulty compound at end of chain
```

Each step lists: trick name + ADD + the modifier added + educational rationale.

**Anchor topology:** `mirage-topology` + `hippy-in-dex` mechanic
**Educational arc:** foundation hippy-in dex → modifier compounds → multi-modifier expansion


**Educational value:** teaches the hippy-in dex mechanic by progressively adding modifiers. Each tier preserves the dex mechanic; only modifier discipline changes.

## Progression rendering rules

1. **Foundation always shown.** Reader sees the anchor 2-3 ADD base before climbing.
2. **Tier ordering by ADD.** Within a progression, lower-ADD tiers come first.
3. **Modifier attribution explicit.** Every step shows what modifier was added and the corresponding ADD increase.
4. **Pilot-tier highlighting.** Pilot members render as links; non-pilot members appear as name-only with explanatory note.
5. **Cross-references to related progressions.** Each progression links to siblings (e.g., the walking-family progression links to the matador-on-butterfly progression).
6. **Modifier-stub filtering.** Modifier-stub rows excluded.

---

## How progressions complement the existing prose

The 98 SCALE pilot rows each carry `prerequisite_notes` field with prose recommendations. Progressions extend this in two ways:

1. **Linear chain visualization.** Where a prose row mentions sibling tricks ad-hoc, progressions render a structured chain.
2. **Cohort-spanning view.** Where prerequisite_notes typically reference 1-3 sibling tricks, progressions can span 5-10 rows in a single educational pathway.

Progressions are NOT a replacement for prose; they're a navigation aid that complements it.
