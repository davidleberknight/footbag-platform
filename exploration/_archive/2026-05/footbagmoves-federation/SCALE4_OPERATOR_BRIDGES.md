## Operator-bridge promotion criteria

For each candidate, the assessment criteria are:

1. **Bridge expansion** — does this row add a new family-base to an existing
   pilot modifier bridge?
2. **Foundation status** — is this a base trick or foundational compound
   that anchors a family tree?
3. **Risk** — pt12 conflict, FM-vocab modifier dependency, or ADD-math gap?
4. **Pedagogical pairing** — does the row pair cleanly with an existing
   pilot row for cross-modifier comparison?

A LOW-risk row scoring well on (1)+(2)+(4) is an SCALE-5 candidate; rows
scoring on (3) are deferred until conflict resolution.
