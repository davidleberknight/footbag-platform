### Part A.1 — Proofs for every evidence-derived ruling

**A1.1 Alpine = the ducking prefix.** Nine independent PassBack folk pairs equate "Alpine X"
with a ducking/diving-augmented tech name: Goliath (Pixie Ducking far DLO = Alpine Smog; smog
= pixie DLO, canonical), Id (Atomic Ducking far Double Down = Alpine Fusion; fusion = atomic
double-over-down, canonical), King Koopa (= Alpine Super Mario; super mario = spinning
symposium torque, canonical), Maelstrom (= Alpine Spinning Whirl), Morpheus (= Alpine Void),
Ripped Warrior (Stepping Ducking Butterfly = Alpine Ripwalk; ripwalk = stepping butterfly,
canonical), Skullsmasher (Atomic Ducking far Mirage = Alpine Atomsmasher; atom-smasher =
atomic mirage, canonical), Superdeeduperfly (= Alpine Superduperfly), Jackknife (Stepping
DIVING ss Butterfly = Alpine Sidewalk; sidewalk = stepping butterfly, canonical).
*Adversarial check passed:* the competing reading "alpine = far" fails, because Ripped
Warrior and Superdeeduperfly carry ducking but NO far/ss token, isolating alpine = +ducking
exactly. *Residual risk:* Jackknife is the one diving (not ducking) case, 8-vs-1; ducking and
diving are both +1 body modifiers in the same four-way family, so ADD is unaffected either
way. Ruling: alpine = +ducking, diving variance noted per row.

**A1.2 Motion is documentarily defined, not derived.** FM carries a bare `Motion` notation
(high confidence): `Toe >> Same Out (DEX) >> Same Out (DEX) > (back) Spin (BOD) > Same Clip
(XBD)(DEL)` — 5 brackets (an earlier draft of this proof miscounted 4; the implementation
review caught it). Three FM compound notations are consistent with it: Locomotion (stepping
prefix + motion body, 6), Motion Sickness (spinning prefix + motion body, 6), Atomotion
(atomic prefix + motion body with [XDEX]). A third source converges: PassBack's tech name
for Motion is "Double Down near Osis", and the FM body IS two out-dexes (the double down)
into a spin + cross-body clipper catch (the osis tail) — the identity that had been parked
as "osis-gated" resolves concretely here. *Adversarial checks passed:* (a) no active
canonical row shares this body, so motion is not an existing trick under another name;
(b) the compounds' first-dex parity varies OP/SAME by prefix, the standard prefix-induced
flip, best-effort refinable without ADD impact. Ruling: register motion as a 5-ADD base per
FM; the compounds author mechanically.

**A1.3 The "screw" suffix.** FM `Torquescrew` (high): `Clip >> Op In (DEX) > Op Back Whirl
(DEX) > (front) Spin (BOD) > Op Clip (XBD)(DEL)` = torque's chassis with one inserted
`Op Back Whirl (DEX)` (+1, 5 total). FM `Moby Dick` (= Mobiusscrew) shows the same inserted
back-whirl dex inside mobius's chassis (+1, 6 total). Two independent notations, one
consistent residue. Ruling: screw = an inserted back-whirl dex, +1; Moby Dick authors from
its own FM notation regardless.

**A1.4 Blazing is +1 by arithmetic.** All eight canonical blazing compounds satisfy
base + 1 exactly: butterfly 3+1=4, drifter 3+1=4, illusion 2+1=3, legover 2+1=3, mirage
2+1=3, paradox_whirl 4+1=5, symposium_mirage 3+1=4, torque 4+1=5. The ADD question is
closed by the corpus; what remains for Red is only the movement/notation definition.

**A1.5 Slapping and flailing get confirm/deny hypotheses, not rulings.** FM defines bare
`Slap` (`Toe > Op In (DEX) >> Same In (DEX)(XDEX) > Op Toe (DEL)`), and FM documents
`Flail = Symposium Illusion` (matching the canonical flail row). But NO compound notation
for a "Slapping X" or "Flailing X" exists to cross-check a prefix reading, so unlike motion
(A1.2) the operator bodies cannot be verified against a second data point. Both stay RED,
downgraded from open questions to single confirm/deny hypotheses in the email.

**A1.6 Monster / Rotor / Wauxspin downgraded to curator-held.** Monster ("Symp. Swirling
Gyro Inside") and Rotor ("Swirling Gyro Inside Stall") are a consistent pair, but no
canonical swirling+gyro stack exemplar exists to mirror a chassis from, and PB gives no dex
counts. Wauxspin ("Butterfly Gyro Toe") puts gyro AFTER the base, a suffix position with no
precedent. Readings are recorded; authoring would require invented chassis, so they hold.
