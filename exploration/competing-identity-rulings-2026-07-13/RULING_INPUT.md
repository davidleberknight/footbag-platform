# Competing-identity ruling session — the ten multi-name rows (J2)

Decision input for the curator session over the ten formula-identity-audit rows
dispositioned "identity-blocked" (one name mapping to more than one formula, or
one source chaining nicknames that other evidence assigns different structures).
Per row: the claims, the evidence with its source, the collision with any
existing canonical, and a recommendation. Adopting a recommendation settles the
NAME-to-FORMULA identity and retags the audit ledger; it does not by itself
author or promote a row (authoring is a follow-on slice under the usual gates).

Source shapes: FootbagMoves (FM) rows carry folk name, technical name, ADD, and
a full notation. Shred Global / stanford (SG) rows are name-only chains (a
shorthand code plus the name with parenthetical aliases): SG attests NAME
EQUIVALENCE, never a formula. footbag.org (FB) is the scrape snapshot.

Shared context: `dragon` is canonical (2 ADD, `SET > SAME OUT [XBD] [DEL]`, a
same-side cross-body catch). FM writes its dragon compounds with an OP-side
dragon catch (`Op Dragon (XBD)(DEL)`), a side tension resolved at notation
derivation, not identity. FM's swirl-dragon notations predate the rotational
register; spelling re-derives at authoring.
