# Class-C Blocker-Validation Audit

Validates every cluster the parser-burndown marked "doctrine-blocked (C)" against the latest
rulings, to eliminate **stale blockers** — clusters held as Red-gated even though a later
ruling, a wired equivalence, or the June-14 consolidation already resolved them.

**Verdict scheme (per the request):**
- **A — Truly doctrine-blocked** (a Red ADD/structure ruling genuinely gates promotion)
- **B — Naming/governance only** (structure settled; only an alias-vs-distinct/presentation call)
- **C — Identification only** (folk name maps to an already-settled structure; no ruling)
- **D — Settled but awaiting authoring** (ADD + decomposition settled; only notation unwritten)

**Authority used:** the June-14 consolidated decision packet
(`exploration/red-frontier-doctrine-round-2026/PACKET.md`) is the current authoritative
open-question list — it states "routine promotions, aliases, notation authoring … proceed
mechanically once the rulings land" and puts everything not requiring a ruling in Appendix B.
A cluster absent from its 8-item checklist is, by that document's own scope, not Red-gated.

1. **Question.** Is DOD (Double-Over-Down, 4 ADD) the same structure as DDD (Down-Double-Down,
   4 ADD), or distinct? Is "down" a base or a naming layer?
2. **Still-unresolved evidence.** PACKET §A1 **OPEN**; both read 4 ADD yet are kept as separate
   base rows with no collapse rule; the `superfly→superduperfly→superdeeduperfly` naming chain
   is unreconciled with the structural stack.
3. **Later rulings.** None. pt14 (nuclear/atomic) did not touch the down family.
4. **Verdict.** **A.** Highest-leverage real gate. Live ~25 (tracked ~46).

### Weaving — verdict **A (truly blocked)**
1. **Question.** Is weaving an operator (+N), a movement family (+0), or a naming layer?

   compound; no source corpus gives its ADD; registered +1 is an explicit placeholder.
3. **Later rulings.** `weaving-magellan` aliased + `weaving-pigbeater` built provisionally, but
   the operator value itself remains undefined — partial wiring, not a resolution.
4. **Verdict.** **A.** Live ~30.

### Pogo — verdict **A (truly blocked)**

3. **Later rulings.** None — it is a clean ruling-vs-corpus standoff awaiting one decision.

### Blurry — verdict **A (truly blocked), but token-count inflated**
