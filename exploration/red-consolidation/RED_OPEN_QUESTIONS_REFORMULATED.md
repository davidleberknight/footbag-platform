# Red Open Questions, Reformulated

The genuinely-unresolved Red questions rewritten in the symbolic-architecture vocabulary established by NF-2A (operator reference), NF-2B (equivalence chains), the symbolic-analysis pass, and the ADD-conflict audit.

This is NOT a copy of old wording. Each question is reframed to surface the structural pattern beneath the original framing, so Red's ruling closes the underlying ontology question rather than the surface case.

Modern vocabulary used throughout:
- **operational layer**: token-level execution notation (clip / toe / DEX / DEL / BOD / XBD / PDX / XDEX with arrows)
- **semantic layer**: modifier-on-base composition (Atomic Mirage; Stepping Paradox Mirage)
- **intermediate operator**: NF-2A entry (atomic, blurry, quantum, nuclear, barraging, furious, whirling, double, high) that compresses a multi-operator stack into one named modifier
- **decomposition stopping**: where a chain terminates -- at a base, at a Q4-blocked operator, at an unresolved-token
- **equivalence chain**: NF-2B-style identity statement (Drifter = Miraging Clipper; Slaying = Symp Sailing = Symp Pixie Atomic)
- **implicit rotational semantics**: operator whose weight depends on the rotational class of its base (atomic, possibly blurry / barraging / furious)
- **operator compression**: when a folk-name reduces a multi-operator stack to a single token (Sailing for Pixie Atomic; Fury for Furious Paradox Mirage)

## Question Set 1 -- Rotational-bonus generalization

**Reframed:** When IFPA promotes an operator to NF-2A locked status, the operator carries a flat weight. | Trick | Stated | Flat formula | Gap | Pattern |
|---|---:|---|---:|---|

| Blurry Torque | 6 | blurry(+1) + torque(4) = 5 | +1 | blurry on compound rotational |

**Q1.A**: Does `blurry` carry +2 on rotational bases (parallel to atomic pt10), and +1 on non-rotational?
- If yes: blurry-whirl and blurry-torque close cleanly. Food Processor depends on which intermediate operator owns the +2.

**Q1.B**: Does `barraging` carry +2 on compound bases (osis = compound)? Or only when the base is rotational? Or is barraging-osis stated 5 sourced from a deeper decomposition (e.g., barraging + osis-as-whirling+osis-base = barraging + whirling + osis-stall)?

**Q1.C**: Does `furious` carry +1 on non-rotational bases (pt6 derived +2 from rotational mirage; the non-rotational case has no IFPA row yet)? This is forward-looking; closing it unblocks future FM-vocab triage.

**Why this packet:** if the answer is "yes for blurry, barraging, furious all rotational +2 / non-rotational +1", a single ruling closes ~5 currently-open rows + standardizes the formula table. If the answer is "no", `ADD_FORMULA_ASSUMPTIONS.md` needs a structural revision.

## Question Set 7 -- Atomic-set polysemy / implicit operator in IFPA readings

**Reframed:** the audit surfaced a systemic +1 gap between PassBack literal decompositions and IFPA stated values across 11 named compounds (Atomsmasher, Blur, Drifter, Dada Curve, Flurry, Food Processor, Fury, Gauntlet, Montage, Spender, Sumo). The pattern is too consistent to be source error.

**Q7 (single high-leverage question):** what is the structural source of the +1 gap?

A. PassBack's `far` carries +1 ADD (closed by Q6.A if Red rules far=+1).

B. IFPA's `Atomic X` is read as `Paradox Atomic X` per pt10's nuclear-as-paradox-atomic-set framing. So atomic-set-as-modifier is functionally +2 in any IFPA composition (not just rotational bases). PassBack's `Atomic far X` is missing the implicit paradox.

C. IFPA stated values include a 1-ADD "set" or "named-trick" baseline that the literal modifier formula doesn't capture. Stated values rest on dictionary-tier authority, not formula derivation.

D. IFPA's `Atomic` reading varies by trick (Atomsmasher: atomic-as-rotational; Blur: stepping-paradox-mirage already; Drifter: miraging-clipper already). The pattern is incidental, not systemic.

Whichever reading wins, the analyzer needs to encode it. The current formula table can't reproduce IFPA stated values for these 11 tricks. Reading B is the most consistent with the pt10 nuclear=paradox+atomic ruling and would also explain why Atomic-on-Mirage gives 4 (atomic-with-implicit-paradox = +2 → 2 + 2 = 4) rather than 3.
