Freestyle operators — reference

Tier-1 operator vocabulary · 14 entries

The fourteen operators below are the primitives of freestyle composition. Each operator combines with a base trick to produce a new named trick; multiple operators stack.

Glyph Name Action Example composition
Set operators  ·  what sends the bag into the air
PIX Pixie Compressed uptime set, leg drives bag through tight orbit PIX + BUTTERFLY → DIMWALK
AT Atomic Set wrapped with full-body rotation in the air AT + OSIS → FLUX
Q Quantum Set with added rotation through uptime Q + MIRAGE → TOE BLUR
BL Blender Set with low-orbit pre-set sweep into the dex BL + BUTTERFLY → BLENDER BUTTERFLY
FAIRY Fairy Set carrying an extra revolution through uptime FAIRY + BLUR → DOUBLE FAIRY
STEP Stepping Foot relocates between set and catch STEP + BUTTERFLY → RIPWALK
Body operators  ·  what the body does while the bag is up
SPIN Spinning Full-body rotation around the vertical axis SPIN + TORQUE → MOBIUS
GY Gyro Body rotation variant (partial / inverted plane) GY + BUTTERFLY → GYRO BUTTERFLY
DUCK Ducking Body drops under the bag mid-dex PIX + DUCK + BUTTERFLY → PHOENIX
PDX Paradox Hip pivot inserted between two dexes PDX + LEG-OVER → PARADOX LEG-OVER
SYMP Symposium Illusion combined with body rotation SYMP + ILLUSION → FLAIL
Structural concepts  ·  relationships across the trick
XDEX Cross-dex Leg circles the bag on the opposite side of the body XDEX + INSIDE → CLIPPER
SAME Same-foot Set foot and catch foot are the same SAME + BUTTERFLY → SAME-FOOT BUTTERFLY
OP Opposite Set foot and catch foot are different (default convention) OP + BUTTERFLY → BUTTERFLY
