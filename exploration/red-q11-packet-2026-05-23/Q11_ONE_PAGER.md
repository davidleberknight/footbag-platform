# Q11 — One-page version

Red,

Across 17 PassBack tricks with clean readings, **the PassBack total is
always lower than what you'd get from adding up the IFPA-side weights
of the parts** — never the same, never higher. Gaps run +1 to +5.

That's too consistent to be random.

**Question:** is PassBack the same scoring system as IFPA, or is it
actually a different convention?

**Four ways to read it:**

- **A.** PB is a **baseline / floor** scoring layer — community-tournament
  scoring; never intended to add up like IFPA-grammar.
- **B.** PB **rounds down deliberately** — a conservative "minimum credit"
  reading.
- **C.** PB uses a **different formula** — maybe different per-modifier
  weights, maybe diminishing returns on stacked modifiers, maybe folk
  names are treated as atomic.
- **D.** PB is a **snapshot from an earlier era** — the gap is timeline
  drift (PB predates pt8 / pt10 / pt12-style rulings).

If A or D: we publish the IFPA-add-up as canonical and surface PB as
"historical / provenance" alongside.
If B or C: we need per-modifier metadata to honor PB's convention
where it applies.

**Sample rows:**

| Trick | PB reading | PB | IFPA-add-up | Gap |
|-------|-----------|----|----|----|
| Blurrage | Stepping far Barrage | 3 | 4 | +1 |
| Predator | Atomic far DLO | 3 | 4 | +1 |
| GDLO | Gyro DLO | 2 | 4 | +2 |
| Pandora's Box | Gyro Pickup | 1 | 3 | +2 |
| Darkwalk | Pixie Diving near Butterfly | 2 | 5 | +3 |
| Colossus | Spinning Diving near Symp. Whirl | 1 | 6 | +5 |

Pick A / B / C / D, write a fifth, or say "not sure" — all answers
help us.
