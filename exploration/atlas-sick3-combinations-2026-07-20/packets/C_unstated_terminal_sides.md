# Review packet C: eleven canonical rows whose terminal delay states no contact side

Self-contained. Question owner: curator (notation authoring). All eleven rows are frozen
until ruled; none appears in any resolved historical Sick 3 sequence, so this is catalog

## The gap

operational notation, so the transition model cannot say whether the ending foot is the
setting foot. The rows fall into four natural groups, and most need a CONVENTION, not a
side: their terminal surface is midline or two-limbed, so "same or opposite foot" does
not apply. Recommending one explicit reading per row:

## Recommended ruling shape

1. Adopt one explicit convention token (or documented reading) for side-inapplicable
   terminals: ten of the eleven rows are midline, two-limbed, or multi-object, where the
   preserve/switch question has no answer because there is no single terminal foot. The
   transition model would represent them as chain-terminating (no carried foot state),
   which matches physical reality: you do not flow from a mouth catch or a two-thigh
   squeeze into a next trick's set the way you do from a toe delay.
2. Author `squeeze` explicitly as either-side (the existing canonically-either notation
   pattern), making it VARIABLE rather than UNRESOLVED.
